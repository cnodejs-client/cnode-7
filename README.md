#### 如何运行? (how to make it work?)
#### 下载项目代码后 (download this project)
* npm i安装依赖(install modules)
* npm start启动
* 打开浏览器,输入localhost:3000即可(open your borwser, input localhost:3000. otherwise you can wait to browser auto open window)
